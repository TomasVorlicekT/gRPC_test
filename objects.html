Name Description
Structure Core
Material Defines the material properties used in structural elements.
Section Specifies the cross-sectional profile for elements like members.
Thickness Defines the thickness of surfaces.
Node Represents a point in space, typically defined by coordinates (e.g., x, y, z).
Line Represents a linear element.
Member A structural element, used to model beams, columns, etc.
Surface Represents a plane bounded by lines.
Opening Defines an opening within a surface, like a window or door in a wall.
Solid Represents a 3D volume element.
LineSet A collection of lines that can be grouped and manipulated as a unit.
MemberSet A collection of members grouped together.
SurfaceSet A collection of surfaces grouped together.
SolidSet A collection of solids grouped together.
Structure Advanced
Block A block represents a model component with typified properties.
Intersection Defines the intersection between structural elements.
LineRelease Allows a line to separate from connected elements under certain conditions.
NodalRelease Defines specific conditions where nodes can release forces or moments.
ResultSection Defines sections where results can be evaluated.
RigidLink Connects structural elements with a rigid link that transmits forces and moments.
StructureModification Applies modifications to structural elements, such as stiffness adjustments.
SurfaceRelease Defines conditions for surface separation from other elements.
SurfaceResultsAdjustment Adjusts the results for surfaces under specific conditions.
SurfacesContact Defines contact interactions between surfaces, such as friction or gap conditions.
